# Blazor.SvgDiagram.Server

	Задача сделать небольшую работу с svg
* в интерфейсе задать размер рабочей зоны и отрисовать
* отобразить квадратную сетку 
* добавить пару компонент - прямоугольник, круг, линия, изображение
* выбор компонента и возможность мышкой переместить. 
	В интерфейсе показываем - выбран круг, размер, позиция. Если переместили, то в интерфейсе для выбранного компонента информация обновляется.

[Библиотека js] (https://github.com/svgdotjs, https://svgjs.dev/docs/3.2/getting-started/)
